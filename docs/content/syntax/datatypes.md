Data types used

## URL

```json
{
    http://www.delven.io
}
```

## String

```json
{
    "key" : "String Value"
}
```

## Numeric

```json
{
    "key" : 12345
}
```

## Boolean

```json
{
    "key" : true | false
}
```





## Object of Key:Value

```json
{
    "params" :
    {
            "p1": "valueA",
            "p2": "valueB",
    }
}
```
 

## Array of Strings

```json
{
    "headers": 
    [
        "Accept: application/xml", 
        "Content-Type: application/xml"
    ]
}
```
